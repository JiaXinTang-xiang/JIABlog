# 新增点击特效

为了提升用户体验和视觉反馈，我为博客添加了鼠标点击特效。

## 新增功能

### 1. 点击水波纹效果
- 点击页面任意位置时显示蓝色水波纹扩散效果
- 支持深色模式适配
- 自动清理，不影响性能

### 2. 彩色粒子效果
- 点击时产生彩色粒子飞散动画
- 10种不同颜色，随机选择
- 流畅的动画过渡

## 技术实现

使用纯 CSS 动画和 JavaScript 实现：
- CSS `@keyframes` 定义动画
- 动态创建和移除 DOM 元素
- 优化性能，避免内存泄漏

这些特效让博客的交互更加生动有趣，同时保持了良好的性能表现。
